一、axios的两种回调

第一种：匿名函数

then(function(data){
    console.log(data);
})

第二种：箭头函数

then((data) =>
 {
    console.log(data)
})

匿名函数和箭头函数的区别：

  • 匿名函数的this指针指向->函数操作的本身
  • 箭头函数的this指针指向->组件

所有要想回显数据到组件，得用箭头函数

二、mounted函数

生命周期函数和浏览器渲染过程挂钩

生命周期函数 是否获取dom节点 是否获取data 是否获取methods
beforeCreate 否 否 否
created 否 是 是
beforeMount 否 是 是
