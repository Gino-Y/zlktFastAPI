vue3.0

一、环境搭建

  1. nvm安装(Node Version Manager)

    1. nvm的作用

      1. nvm是用来用来管理nodeJs版本的。 假设有多个项目，每个项目依赖的node版本不一样，这个时候就可以用到nvm来管理node版本

    2. 安装

      1. 下载地址：https://github.com/coreybutler/nvm-windows/releases ，

      2. win下载nvm-setup.zip即可

      3. 双击nvm-setup.exe安装即可

    3. 配置环境变量

      1. 把nvm安装的路径添加到path路径下即可

    4. 验证

      1. cmd中运行nvm命令

    5. nvm的常用命令

      1. nvm install [version]：安装指定版本的node.js

      2. nvm use [version]：使用某个版本的node

      3. nvm list：列出当前安装了哪些版本的node

      4. nvm uninstall [version]：卸载指定版本的node

      5. nvm node_mirror [url]：设置nvm的镜像

      6. nvm npm_mirror [url]：设置npm的镜像

  2. node安装

    1. 安装指定版本的node

      1. nvm install 10.16.0

    2. 设置nvm源

      1. nvm node_mirrorhttps://npm.taobao.org/mirrors/node/设置nvm的镜像。

      2. nvm npm_mirrorhttps://npm.taobao.org/mirrors/npm/设置npm的镜像。

二、npm的使用

npm已经在安装Node的时候顺带装好了

  1. 作用

    1. NodeJs的包管理工具

    2. npm可以根据每个包之间的依赖关系，把所有依赖的包都下载下来并管理起来

  2. 常用命令

    1. nvm use 10.16.0(使用node后才可以使用npm)

    2. 查看版本：npm -v

    3. 初始化：npm init 进入指定项目目录执行，会生成一个package.json文件，保存本项目中用到的包

  3. 安装需要的包

    1. npm install vue # 本地安装

    2. npm install vue -g # 全局安装

    3. 注意：--save参数会将安装的包记录到package.json中

  4. 使用淘宝镜像

    1. 安装cnpm：npm install -g cnpm --registry=https://registry.npm.taobao.org

    2. 以后就可以使用cnpm来安装包

  5. 安装vue/cli

    1. cnpm install -g @vue/cli #全局安装vue-cli

三、vue-cli脚手架

vue-cli是一个vue的工具，可以快速构建vue项目，并且创建好了一下脚手架代码，很方便

  1. vue-cli安装：工具没必要每个项目都安装一次，所以安装到全局即可
    1. npm install -g @vue/cli
  2. vue-cli创建项目
    1. 在存放项目路径下执行：vue create [项目名称]
    2. 使用镜像：vue create -r https://registry.npm.taobao.org [项目名称]

  3. 项目结构介绍

    1. node_modules：本地安装的包的文件夹
    2. public：项目出口文件

    3.src：项目源文件：

    1. assets：资源文件，包括字体，图片等

    2. components：组件文件

    3. App.vue：入口组件

    4. main.js：webpack在打包的时候的入口文件

    4.babel.config.js：es*转低级js语言的配置文件。

    5.package.json：项目包管理文件

    注意：vscode安装vetur包用来识别.vue文件

  4. 启动项目

    1. 可以在package.json文件中找到命令

      npm run serve
