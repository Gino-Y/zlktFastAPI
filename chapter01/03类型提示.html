类型提示

1.什么是类型提示？有什么作用？

Python 是动态类型语言，运行时不需要指定变量类型，这就会出现一个问题，如果不知道类型，用的时候就没法使用类型下的一些函数等，比如字符串的split

示例：

def split_str(s):
    strs = s.split(",")


参数str由于不知道什么类型，所以没有split代码提示，只有当你传了字符串才会正确执行，整型等会报错

2.怎么才能使用类型提示？

name:str

示例：

def split_str(s:str):
    strs = s.split(",")
