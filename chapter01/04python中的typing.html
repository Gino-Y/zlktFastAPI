python中的typing

可用于类型检查器、IDE、静态检查器等第三方工具。

文档地址：

  • https://mypy.readthedocs.io/en/latest/cheat_sheet_py3.html
  • https://docs.python.org/zh-cn/3/library/typing.html#type-aliases 
int,long,float: 整型,长整形,浮点型;

bool,str: 布尔型，字符串类型；

List, Tuple, Dict, Set:列表，元组，字典, 集合;

Iterable,Iterator:可迭代类型，迭代器类型；

Generator：生成器类型；


注意首字母大小写

二、常用类型示例

1.指定参数类型

def greeting(name: str):
    retuen "hello"

2.指定函数返回的参数类型

一种类型的：
    def greeting(name: str) -> str:
        return "hello"

多种类型的：
    def greeting(name: str,num:int) -> List[int or str]:  # 或者Union[int, str]
        list1 = []
        list1.append(name)
        list1.append(num)
        return list1

3.类型别名，可以讲复杂一点类型给个别名，这样好用一些

from typing import List
Vector = List[float]

ConnectionOptions = Dict[Vector, str]

4.NewType，可以创建不同的类型

  • 用于向类型检查器指示不同的类型
  • NewType(name, tp)返回一个函数，这个函数返回其原本的值。静态类型检查器会将新类型看作是原始类型的一个子类
from typing import NewType

UserId = NewType('UserId', int)
some_id = UserId(524313)



def name_by_id(user_id: UserId) -> str:
    pass


调用：
    name_by_id(UserId(42))  正确
    name_by_id(42)          错误
    UserId('user')          错误

5.Callable：查看是否可调用

查看是否可调用：
    isinstance(函数名称, Callable)    # 返回True或False




Callable作为参数：
    from typing import Callable

    def print_name(name:str):
        print(name)

    def get_name(get_next_item: Callable[[str],None]):
        return get_next_item

    a = get_name(print_name)
    print(a("hallen"))

Callable作为返回值
    def print_name(name:str):
        print(name)

    def ret_name() -> Callable[[str],None]:
        return print_name

6.泛型

from typing import TypeVar

t = TypeVar("T")

def get_data(a:t):
    print(a)

get_data(11)
get_data("aa")

7.自定义泛型类

T = TypeVar('T')

class UserInfo(Generic[T]):   # 继承Generic[T]，UserInfo[T]也就是有效类型
    def __init__(self,v:T):
        self.v = v

    def get(self):
        return self.v

l = UserInfo("hallen")

print(l.get())

注意：Generic 类型变量的参数应各不相同

T = TypeVar('T')
A = TypeVar('A', str, bytes)


Generic[T, T]  这种不行

Generic[T, A]  这种可以

8.Any类型

from typing import TypeVar,Any

t = TypeVar("T")

def get_data(a:Any):
    print(a)

get_data(11)
get_data("aa")

可对 Any 类型的值执行任何操作或方法调用，并赋值给任意变量,Any和Object类型
