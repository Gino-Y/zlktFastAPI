知了fastapi从入门到项目实战

get请求

1.路径参数：http://127.0.0.1:8000/user/1

1.路径参数
@app.get("/user/{user_id}")
def info(user_id):
    return {"user_id":user_id}


2.限定参数类型,在函数参数中限定类型

@app.get("/user/{user_id}")
def info(user_id:int):
    print(type(user_id))    # 类型是int类型
    return {"user_id":user_id}


3.路径转换器：将一个路径作为参数，包含/的路径，比如：/张三/18/xxx，姓名、年龄，住址都传过去作为参数
@app.get("/user/{info:path}")   #这里必须叫path
def user_info(info:str):
    return info

4.枚举:限定路径必须为枚举中的：http://127.0.0.1:8000/models/alexnet
    from enum import Enum
    from fastapi import FastAPI

    class ModelName(str, Enum):
        alexnet = "alexnet"
        resnet = "resnet"
        lenet = "lenet"


    app = FastAPI()

    @app.get("/models/{model_name}")
    async def get_model(model_name: ModelName):
        if model_name == ModelName.alexnet:
            return {"model_name": model_name, "message": "Deep Learning FTW!"}

        if model_name.value == "lenet":
            return {"model_name": model_name, "message": "LeCNN all the images"}

        return {"model_name": model_name, "message": "Have some residuals"}

注意：

  • 这种必须传参，否则找不到url
  • 路径参数的顺序很重要，顺序决定后端接收的值

2.查询参数

http://127.0.0.1:8000/get?name=xxx&age=18

http://127.0.0.1:8000/get?age=18&name=xxx

@app.get("/get")
def get(name:str,age:int):
    return {"name":name,"age":age}

注意：

  • 这种必须传参，否则找不到url，可以指定默认值解决这个问题，但是默认值不一定是想要的，慎重
    • name:str="",age:int=0
  • 查询参数的顺序不重要，值由对应的key决定的，哪个在前哪个在后无所谓

3.可选参数

  • 传了我用，不传也不影响url的访问
  • 参数的默认值指定为None即可
  • 路径参数没有默认值
  • 查询参数才有默认值
@app.get("/get")
def get(name:str=None,age:int=None):
    if name and age:
        return {"name": name, "age": age}

    return "hello"

4.路径参数和查询参数都有

路径参数

查询参数，并且查询参数可以不用传的，使用默认值None

@app.get("/users/{user_id}/article/{a_id}")
def read_user_item(
    user_id: int, a_id: int, a_name: str = None, readed: bool = False
):
    item = {"a_id": a_id, "user_id": user_id,"a_name":a_name,"readed":readed}

    return item

属性具有默认值时，它不是必需的。否则它是一个必需属性。

将默认值设为None可使其成为可选属性

注意：

  • 如果路径参数和查询参数都有，则路径参数必须在查询参数前面

5.数据校验

1.查询参数：Query

  • 使用
    • 第一个参数是默认值，如果是必传参数，则第一个参数必须为...
    • 第二个及之后的参数为设置项
name:str = None        可选参数
from fastapi import FastAPI, Query
name:str = Query(None)   和前面的效果一样，但可以继续扩展
name:str = Query(None, max_length=50)   可选参数，但是只要传参了，长度必须小于50

  • 其他设置项

    • min_length：最小长度

    • max_length：最大长度

    • regex：正则

    • title：有关参数的信息

    • description：有关校验的描述

    • alias：别名，用于校验在python中命名不可用的参数，比如user-name，中划线这种python不能读取，可以用alias指定，alias=“user-name”,就会读取到该参数

    • deprecated，值为True或False，True表示弃用，会在api文档中体现出来，该参数将被弃用

    • gt：大于

    • lt：小于

    • ge：大于等于

    • le：小于等于

2.路径参数：Path

  • 1.使用
    • 第一个参数是... 因为路径参数总是必需的，它必须是路径的一部分。所以第一个参数为三个点
    • 第二个及之后的参数为设置项
from fastapi import FastAPI, Path, Query
item_id: int = Path(..., title="The ID of the item to get")
  • 2.其他设置项
    • 和Query类似
