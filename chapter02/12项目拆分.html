知了fastapi从入门到项目实战

项目拆分

类似与flask中的蓝图

一、项目结构说明：

  • fastapi_test：项目名
  • main.py：主程序
  • app：所有的应用
    • article：article模块，必须是个python package
      • router：article的分路由
    • user：user模块，必须是个python package
      • router：user的分路由

二、分路由

1.article/router.py

from fastapi import APIRouter

router = APIRouter(
    prefix="/article"          # 相当于路由分组的前缀
)


@router.get("/list")          # 路由为：/article/list
def get_articles():
    return []

2.user/router.py

from fastapi import APIRouter


router = APIRouter(
    prefix="/user"            # 相当于路由分组的前缀
)

@router.get("/info")          # 路由为：/user/info?user_id=3
def get_user(user_id:int):
    return user_id

三、主程序中引入分路由：main.py

from fastapi import FastAPI
# 导入分路由，注意别名
from app.article import router as article_router
from app.user import router as user_router

# 注意这里是FastAPI，不是APIRouter
app = FastAPI()


# 导入分路由
app.include_router(article_router.router)
app.include_router(user_router.router)

# 还可以加路由前缀，比如管理端和前端的区分
# 还可以加依赖dependencies[Depends(get_token_header)]
# app.include_router(user_router.router,prefix="/admin")
# app.include_router(user_router.router,prefix="/front")



# 主路由url映射
@app.get("/")
def hello():
    return {"msg":"hello fastapi"}

四、子应用程序

from fastapi import FastAPI

app = FastAPI()


@app.get("/app")
def read_main():
    return {"message": "Hello World from main app"}


subapi = FastAPI()


@subapi.get("/sub")
def read_sub():
    return {"message": "Hello World from sub API"}


app.mount("/subapi", subapi)
