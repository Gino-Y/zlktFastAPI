依赖注入

Depends，依赖注入

一、什么是依赖注入

将其他组件与FastAPI集成

在程序中我们的代码可以声明一些它必须依赖的项：我们称之为dependencies，也就是依赖项

FastAPI会把所有需要的依赖项提供给你的代码，称之为"注入"依赖项

二、依赖注入有什么作用

  1. 业务逻辑复用的场景使用，可以减少重复代码
  2. 共享数据库连接
  3. 强制执行安全性、身份验证、角色管理等
  4. 其他使用场景

三、简单的依赖注入

from fastapi import Depends

# 返回共有信息
def get_common_data(limit:int=0):
    return {"limit":limit}

@app.get("/get_articles")
def get_articles(common_data:dict= Depends(get_common_data)):
    return common_data

@app.get("/get_users")
def get_users(common_data:dict= Depends(get_common_data)):
    return common_data


一些共有信息可以在依赖函数中封装好，然后注入就可以，效果和继承类似
还可以做权限校验等

注意：依赖项可以接收路径参数和查询参数，查询参数在依赖注入返回值前面

流程：

  • 使用正确的参数调用您的依赖项（“可靠”）函数。
  • 从您的函数中获取结果。
  • 将该结果分配给路径操作函数中的参数

四、类依赖注入

class GetCommonData:
    def __init__(self,limit):
        self.limit = limit


@app.get("/get_articles")
def get_articles(common_data:GetCommonData= Depends(GetCommonData)):
    return common_data.limit

@app.get("/get_users")
def get_users(common_data = Depends(GetCommonData)):
    return common_data.limit

注意：

  • 类的参数在init函数中传递，和函数一样的道理
  • 注入返回的类型就是类：common_data:get_common_data（或者不申明类型也一样）
common_data:GetCommonData = Depends()
common_data:GetCommonData= Depends(GetCommonData))
common_data = Depends(GetCommonData))
上面三个效果一样，建议使用第一个

五、子依赖

def get_limit():
    return 10

# 返回共有信息
def get_common_data(limit:int = Depends(get_limit)):
    return {"limit":limit}

@app.get("/get_articles")
def get_articles(common_data:dict= Depends(get_common_data)):
    return common_data

只需注意返回的数据类型，这个例子没有实际意义

六、list列表依赖注入（多依赖对象注入）

from fastapi import Depends, FastAPI, Header, HTTPException

def verify_token(x_token: str = Header(...)):
    if x_token != "fake-super-secret-token":
        raise HTTPException(status_code=400, detail="X-Token header invalid")


def verify_key(x_key: str = Header(...)):
    if x_key != "fake-super-secret-key":
        raise HTTPException(status_code=400, detail="X-Key header invalid")
    return x_key

@app.get("/items/", dependencies=[Depends(verify_token), Depends(verify_key)])
def read_items():
    return [{"item": "Foo"}, {"item": "Bar"}]

dependencies不接受返回值，要接受返回值就用普通的依赖

七、全局依赖

向整个应用程序添加依赖项，它们将应用于应用程序中的所有路径操作：

app = FastAPI(dependencies=[Depends(verify_token), Depends(verify_key)])

八、yield

在完成后执行一些额外步骤的依赖项，就要使用yield，而不是return

1.安装包，3.7及以上版本不需要

pip install async-exit-stack async-generator

2.使用

def get_db():
    db = DBSession()
    try:
        yield db    # 响应传递后执行语句后面的代码
    finally:        # 确保后面的代码一定会执行
        db.close()
