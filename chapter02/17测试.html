测试

一、pytest

官方文档：https://docs.pytest.org/en/latest/contents.html

知了课程：https://study.163.com/course/introduction.htm?courseId=1210816809#/courseDetail?tab=1

1.安装包

pip install pytest

2.规则

  • 所有的单测文件名都需要满足test_*.py格式或*_test.py格式
  • 测试函数以test_开头

  • 测试类以Test开头，并且不能带有 init 方法

  • 断言使用基本的assert即可

3.使用

'''
@IDE ：PyCharm
@Author ：知了-海龙
@Date ：2021/6/30 10:48
@Email：hallen200806@163.com
@Desc ：
'''

import pytest

def test_a():
    print("==========testa")
    assert 1

def test_b():
    print("==========testb")
    assert 0

if __name__ == '__main__':
       pytest.main(["-s","test_pytest.py"])

4.pytest.main参数

-m=xxx: 运行打标签的用例
-q: 安静模式, 不输出环境信息
-v: 丰富信息模式, 输出更详细的用例执行信息
-s: 显示程序中的print/logging输出
--resultlog=./log.txt 生成log
--junitxml=./log.xml 当前目录下生成xml报告
--html=./report.html' 当前目录下生成html报告


举例：
    pytest.main(["目录名"])   运行当前目录下所有满足条件的测试
    pytest.main(['test_xxx.py'])  运行单个测试文件
    pytest.main(['test_reg.py::类名::函数名']) 运行指定的类和指定的函数

5.运行测试

if __name__ == '__main__':
    pytest.main(["-s","test_pytest.py"])

激活虚拟环境,cmd运行：pytest ./test_pytest.py

激活虚拟环境,cmd运行：pytest

二、fastapi测试

使用对象requests

1.使用

from fastapi import FastAPI
from fastapi.testclient import TestClient


app = FastAPI()


@app.get("/")
async def read_main():
    return {"msg": "Hello World"}


client = TestClient(app)

def test_read_main():   # 函数名必须test_开头，这是pytest的约定
    response = client.get("/")
    assert response.status_code == 200
    assert response.json() == {"msg": "Hello World"}


# 测试websocket   
def test_websocket():
    with client.websocket_connect("/ws") as websocket:
        data = websocket.receive_json()
        assert data == {"msg": "Hello WebSocket"}

2.分离

main.py

from fastapi import FastAPI

app = FastAPI()


@app.get("/")
async def read_main():
    return {"msg": "Hello World"}

test.py

from fastapi.testclient import TestClient

from .main import app

client = TestClient(app)


def test_read_main():
    response = client.get("/")
    assert response.status_code == 200
    assert response.json() == {"msg": "Hello World"}

3.运行

pip install pytest


然后运行pytest，会自动检测以test_开头的测试函数

四、request请求

1.使用

from fastapi import FastAPI, Request

@app.get("/aa")
def read_root(request: Request):   # 必须是参数，不然没法获取客户端信息
    client_host = request.host
    return client_host

2.常用方法

request.url：请求的url
request.method：请求的方法
request.headers：请求头信息
request.query_params["name"]：获取查询参数
    http://127.0.0.1:8000/aa?name=hallen

request.client.host：获取客户端host

request.cookies.get('mycookie')：获取cookie
