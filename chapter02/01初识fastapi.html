知了fastapi从入门到项目实战

初识fastapi

https://fastapi.tiangolo.com/zh/tutorial/

一、环境安装

uvicorn作为服务器

安装所有的依赖：
    pip install fastapi[all]        建议使用

分步安装：
    pip install fastapi
    pip install pydantic
    pip install uvicorn[standard]

二、使用

1.导入

from fastapi import FastAPI

2.实例化FastAPI

app = FastAPI()


参数：
    debug: bool = False,
    routes: Optional[List[BaseRoute]] = None,
    title: str = "FastAPI",
    description: str = "",
    version: str = "0.1.0",
    openapi_url: Optional[str] = "/openapi.json",
    openapi_tags: Optional[List[Dict[str, Any]]] = None,
    servers: Optional[List[Dict[str, Union[str, Any]]]] = None,
    dependencies: Optional[Sequence[Depends]] = None,
    default_response_class: Type[Response] = Default(JSONResponse),
    docs_url: Optional[str] = "/docs",
    redoc_url: Optional[str] = "/redoc",
    swagger_ui_oauth2_redirect_url: Optional[str] = "/docs/oauth2-redirect",
    swagger_ui_init_oauth: Optional[Dict[str, Any]] = None,
    middleware: Optional[Sequence[Middleware]] = None,
    exception_handlers: Optional[
        Dict[
            Union[int, Type[Exception]],
            Callable[[Request, Any], Coroutine[Any, Any, Response]],
        ]
    ] = None,
    on_startup: Optional[Sequence[Callable[[], Any]]] = None,
    on_shutdown: Optional[Sequence[Callable[[], Any]]] = None,
    openapi_prefix: str = "",
    root_path: str = "",
    root_path_in_servers: bool = True,
    responses: Optional[Dict[Union[int, str], Dict[str, Any]]] = None,
    callbacks: Optional[List[BaseRoute]] = None,
    deprecated: Optional[bool] = None,
    include_in_schema: bool = True,

3.使用实例化后的app对象映射路由

@app.get("/get")
@app.post("/post")
@app.put("/put")
@app.delete("/delete")


@app.options("/options")
@app.head("/head")
@app.patch("/patch")
@app.trace("/trace")

注意：

  • 必须以 "/" 开始
  • 必须是装饰器
    • 是旧函数代码不变的情况下增加功能，一般为了简化装饰器会用语法糖@新函数来简化

4.路由对应的业务函数编写

@app.get("/user")
def hello():
    return {"msg":"hello fastapi"}   # return可以返回任意类型的值，比如数字，列表等

5.启动uvicorn服务

uvicorn main:app --reload


main指main.py

app指第二步实例化后的对象名称

--reload的作用是只要改动了代码，会自动识别，然后重新启动，会消耗一定的性能，生产上不建议使用
--host 0.0.0.0 
--port 8080


nohup uvicorn main:app --host 0.0.0.0 --port 8080

6.访问

http://127.0.0.1:8000/user      你自己定义的路由，@app.get("/user")


http://127.0.0.1:8000/docs      swagger接口文档


http://127.0.0.1:8000/redoc     redoc接口文档


http://127.0.0.1:8000/openapi.json  查看json文档
        可以使用它自动生成与你的 API 进行通信的客户端代码，都是基于openapi的

7.直接运行py文件，且修改端口等信息，通过这种方式可以debug

import uvicorn

if __name__ == '__main__':
    uvicorn.run(app='main:app', host="127.0.0.1", port=8000, reload=True, debug=True)


run的其他参数：
    uvicorn/config.py:119

8.api文档

所有的路由默认是生成api文档的，如果不想添加到api文档，可以设置include_in_schema=False

@app.get("/items/", include_in_schema=False)
async def read_items():
    return [{"item_id": "Foo"}]
